stripe-express-checkout-element demo

One-click wallets (Apple Pay / Google Pay / Link). Enter your Stripe test publishable key to render the wallet buttons.

Stripe publishable key Must start with pk_test_. Never use a live key here.

Wallet buttons not showing?
  • The page must be served over HTTPS. StackBlitz serves over HTTPS, so that requirement is already met.
  • Apple Pay only appears in Safari on a Mac or iOS device that has a card saved in Wallet.
  • Google Pay only appears in Chrome signed in to a Google account that has a saved card.
  • Link renders for most browsers in test mode, so it is usually the wallet you will see first.
  • In live mode some wallets require a verified domain in the Stripe Dashboard. Test mode is lenient and does not require domain verification.
  • If no wallet is available for your browser/device, the Express Checkout Element renders nothing — that is expected behavior, not an error.
